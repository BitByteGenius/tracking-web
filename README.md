# 🚀 Live Tracking & Attendance Management System

A full-stack Live Tracking & Attendance Management System built with **Flutter**, **Node.js**, **Express.js**, and **MongoDB**. The system enables employees to securely check in using GPS and selfie verification, automatically tracks live location during working hours, and allows administrators to monitor attendance and employee locations through a responsive dashboard.

## ✨ Features

### 👤 Employee
- Secure Authentication (JWT)
- GPS-based Check-In & Check-Out
- Selfie Verification
- Automatic Live Location Tracking
- Attendance History
- Working Hours Calculation
- Distance Tracking
- Profile & Settings
- Material 3 Responsive UI

### 👨‍💼 Admin
- Secure Admin Login
- Live Employee Tracking
- Attendance Dashboard
- Employee Management
- Online/Offline Status
- Attendance Reports
- Responsive Dashboard
